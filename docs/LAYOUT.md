# 레이아웃 커스터마이징

## 패널

화면은 좌측, 중앙, 우측 패널로 구성됩니다.

- 패널 경계선을 드래그하면 폭이 바뀝니다.
- 변경된 폭은 브라우저 `localStorage`에 저장됩니다.
- 모바일 폭에서는 패널이 세로로 재배치됩니다.

## 위젯

- 각 위젯 왼쪽 상단의 이동 아이콘을 드래그해 다른 패널로 옮길 수 있습니다.
- 위젯 우하단을 드래그하면 크기를 조절할 수 있습니다.
- 탭별 레이아웃은 사용자별 브라우저 `localStorage`의 `cospi-layout-v2-{userId|anon}`에 저장됩니다.
- 위젯 크기는 우하단 리사이즈 동작이 감지될 때만 `cospi-widget-sizes-v2-{userId|anon}`에 저장됩니다.

## 서버 저장

현재 구현은 빠른 로딩을 위해 브라우저 저장을 기본으로 사용합니다. 로그인 사용자는 사용자 ID가 붙은 키로 레이아웃이 분리되며, 서버 DB 저장이 필요하면 `/api/settings`의 `layout` 필드로 확장할 수 있습니다.

## 초기화

브라우저 개발자도구에서 아래 키를 삭제하면 기본 레이아웃으로 돌아갑니다.

- `cospi-layout-v2-*`
- `cospi-panel-widths-v2-*`
- `cospi-widget-sizes-v2-*`
